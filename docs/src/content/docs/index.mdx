## Key Features

<CardGrid stagger>
	<Card title="LSP Semantic Highlighting" icon="code">
		Integrates with Lean's actual parser and LSP server for precise tokenization and native color schemes.
	</Card>
	<Card title="Interactive Proof States" icon="information">
		Hover over turnstile (`⊢`) markers to dynamically inspect hypotheses and active goals at each tactic step.
	</Card>
	<Card title="Synchronized Hovers" icon="random">
		Selects and highlights all occurrences of the same resolved variable or definition across the document.
	</Card>
	<Card title="Recursive Tooltips" icon="puzzle">
		Hover over nested signatures inside tooltip overlays recursively, keeping stack-based tooltips open seamlessly.
	</Card>
</CardGrid>

## Quick Start

Transform your Markdown files with interactive Lean blocks:

```ts
import { remark } from "remark";
import remarkHtml from "remark-html";
import remarkLean from "remark-lean";

const html = await remark()
  .use(remarkLean, { rootUri: "file:///path/to/lean-project" })
  .use(remarkHtml, { sanitize: false })
  .process(markdown);
```

Then load the runtime and stylesheet in your client application:

```ts
import { leanHydrate } from "remark-lean/runtime";
import "remark-lean/lean.css";

// Hydrate Lean interactive elements
leanHydrate();
```
